# hub.js

Through a simple way to deal with the event flow.

## Simple Usage

```js
import $hub from './dist/hub.min.js';

// register an event listener
$hub.listen('test', ( data ) => {
    console.log( 'test', data );
});

setInterval(() => {
    // send the 'test' event
    $hub.emit('test', { code: 1 });
}, 1000);
```
